# Demo 2: Search Tools Build Context — "Read the Room"

## Prompt

What are the main performance bottlenecks in NYC's 311 system, and which agencies are most affected? Ground your analysis in the documents and data available.

## What to Watch

- The agent searches for relevant files rather than being told which to read
- It reads CLAUDE.md to understand the project context
- It cross-references qualitative documents with quantitative data
- No files were specified in the prompt — the agent built its own context
