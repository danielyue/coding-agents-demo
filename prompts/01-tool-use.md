# Demo 1: Tool Use in a Loop — "Watch the Loop"

## Prompt

Read through all the files in engagement-docs/. Produce a structured findings memo with key themes and operational pain points. Write it to FINDINGS.md.

## What to Watch

- The sequence of tool calls: read, read, read, write
- The agent reads all documents without being told which ones matter
- Cross-references between documents emerge in the output
- No code execution needed — just read and write tools
